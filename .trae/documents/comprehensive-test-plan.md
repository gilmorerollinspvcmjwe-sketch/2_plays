# 乙女游戏创作者模拟器 - 综合测试计划

## 测试概述

**项目名称**：乙女游戏创作者模拟器  
**测试目标**：全面验证游戏各功能模块的正确性、稳定性和用户体验  
**测试范围**：前端 Vue3 应用、后端 AI 服务、数据持久化、移动端适配  
**测试类型**：功能测试、集成测试、性能测试、兼容性测试

---

## 一、测试环境

### 1.1 开发环境
- **前端**：Vue 3.4 + TypeScript 5.3 + Vite 5.2
- **UI 库**：Vant 4.8 + TailwindCSS 3.4
- **状态管理**：Pinia 2.1
- **后端**：Node.js + Express
- **AI 服务**：MiniMax、Kimi、DeepSeek、豆包、通义千问

### 1.2 测试设备
- iPhone SE (375px)
- iPhone 12/13/14 (390px)
- iPhone 14 Pro Max (430px)
- Android 主流机型 (360px-414px)
