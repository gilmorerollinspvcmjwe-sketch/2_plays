# 主页功能合并与"下一天"按钮设计计划

## 概述

将【开发】功能合并到【主页】中，并在底部导航栏中央添加一个显眼的"下一天"大按钮，作为游戏的核心驱动机制。

---

## 一、当前结构分析

### 1.1 现有底部导航栏 (MainLayout.vue)
```
┌─────────┬─────────┬─────────┬─────────┐
│  首页   │  开发   │  运营   │  评论   │
│  (0)    │  (1)    │  (2)    │  (3)    │
└─────────┴─────────┴─────────┴─────────┘
```

### 1.2 【开发】页面 (CreatorEntryView.vue) 包含的功能
- **创建角色** - 设计角色形象、性格、背景
- **设计剧情** - 编写故事线、分支、结局
- **剧情分析** - 查看完成率、流失点分析
- **角色排行** - 查看角色价值排名
- **开发进度** - 显示项目进度条和前置条件提示

### 1.3 【我的】页面 (ProfileView.vue) - 这是另一个页面
包含：创作者等级、技能树、成就墙、称号、统计等

## 二、设计方案

### 2.1 底部导航栏改造

**新布局**（4个Tab + 中间"下一天"大按钮）：
```
┌─────────┬─────────┬─────────────┬─────────┐
│  首页   │  运营   │   下一天    │  评论   │
│  (0)    │  (1)    │   (大按钮)  │  (2)    │
└─────────┴─────────┴─────────────┴─────────┘
```

**"下一天"按钮设计**：
- 位置：底部导航栏正中央
- 样式：圆形大按钮，突出显示
- 颜色：渐变色（粉色系 #FF69B4 → #FF1493）
- 图标：日历/时间相关图标（van-icon: clock-o 或 calendar-o）
- 文字："下一天"
- 点击效果：脉冲动画 + 缩放反馈
- 尺寸：70px × 70px（比普通Tab大）
- 位置偏移：向上突出 -20px

### 2.2 【开发】功能合并到【主页】

**合并策略**：
将 CreatorEntryView.vue 的4个开发选项卡片整合到 HomeView.vue 中

**主页新结构**：
```
HomeView.vue
├── 公司数据看板（原有）
├── 快速开发入口卡片（新增）
│   ├── 创建角色
│   ├── 设计剧情
│   ├── 剧情分析
│   └── 角色排行
├── 待处理事项（原有）
```

**详细合并内容**：

| 开发页功能 | 合并方式 | 主页位置 |
|-----------|---------|---------|
| 创建角色 | 入口卡片 | 快速开发入口卡片 |

2. **添加路由跳转方法**

```typescript
const router = useRouter();
const projectStore = useProjectStore();

const currentProject = computed(() => projectStore.currentProject);

// 是否可以开始开发
const canStartDev = computed(() => {
  if (!currentProject.value) return false;
  const hasChars = currentProject.value.characters.length >= 2;
  const hasPlots = currentProject.value.plots.length >= 1;
  const teamSize = currentProject.value.team.planning + currentProject.value.team.art +
                   currentProject.value.team.program + currentProject.value.team.operation;
  return hasChars && hasPlots && teamSize >= 2;
});

function goToCharacterCreator() {
  if (!currentProject.value) {
    showToast('请先选择一个项目');
    return;
  }
  router.push('/creator/character');
}

function goToPlotDesigner() {
  if (!currentProject.value) {
    showToast('请先选择一个项目');
    return;
  }
  router.push('/creator/plot');
}

function goToPlotAnalysis() {
  if (!currentProject.value) {
    showToast('请先选择一个项目');
    return;
  }
  router.push(`/plot-analysis/${currentProject.value.id}`);
}

function goToCharacterRanking() {
  router.push('/character-ranking');
}
```

## 五、数据流设计

### 5.1 "下一天"按钮数据流

```
用户点击"下一天"
    ↓
MainLayout.vue 处理点击
    ↓
调用 operationStore.simulateOneDay()
    ↓
调用 simulationStore.tick()
    ↓
更新所有游戏数据（日期、员工、角色、项目）
    ↓
触发评论生成（如果有运营项目）
    ↓
触发随机事件
    ↓
更新UI显示
    ↓
显示Toast："已进入第X天"
```
