# Tasks - 玩家模拟系统

## 任务说明

本任务是实现玩家模拟系统的核心功能，采用**基于代理的模拟（Agent-Based Simulation）**架构。

## 系统架构

```
玩家点击"下一天" 
    ↓
SimulationEngine.tick() 
    ↓
┌─────────────────────────────────────────────────────────────────┐
│               分层模拟架构                                        │
│  Layer 1: 宏观计算（毫秒级）                                    │
│    - DAU预测、留存预估、收入预估                                │
│                              ↓                                  │
│  Layer 2: 微观模拟（100-500玩家）                               │
│    - 每个玩家独立决策：抽卡/评论/活动/社交                      │
│                              ↓                                  │
│  Layer 3: 关键事件（如需要）                                    │
│    - 极端行为详细模拟、舆情事件追踪                              │
└─────────────────────────────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────────────────────────────┐
│                 对玩家公司/项目/角色/剧情的真实影响               │
└─────────────────────────────────────────────────────────────────┘
```
