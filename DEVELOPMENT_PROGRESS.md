## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问应用
打开浏览器访问 `http://localhost:5173`

### 4. 功能体验
- **首页**：查看欢迎横幅、快速开始、功能入口、每日任务
- **角色创建**：点击"开发"Tab → 选择外貌/服装/性格/背景 → AI 润色
- **剧情设计**：点击"开发"Tab → 选择甜宠/虐恋/悬疑路线 → 编辑章节
- **积分商城**：点击顶部积分图标 → 签到/分享/购买 AI 服务
